COMP 3104 – DevOps is a course offered at George Brown College, focusing on the principles and practices of Development and Operations (DevOps) in modern software development. The course covers continuous integration (CI), continuous deployment (CD), automation, cloud computing, and containerization. Students learn how to streamline software development and IT operations by using tools such as Docker, Kubernetes, Jenkins, GitHub Actions, and cloud platforms like AWS or Azure.

This course emphasizes hands-on learning, allowing students to work on real-world projects that integrate coding, testing, deployment, and monitoring in an automated environment. Through labs and assignments, students develop expertise in infrastructure as code (IaC), scripting, and security best practices. By the end of the course, students will have a strong foundation in DevOps workflows, making them valuable assets in software development and IT industries.DevOps is crucial in modern software development as it enhances collaboration, increases deployment speed, and ensures software reliability. By automating workflows and integrating testing early, DevOps reduces errors, minimizes downtime, and enables faster delivery of high-quality applications, making it an essential practice in today’s tech industry.
